# Zivoe

**Zivoe** 是一款面向儿童骑行产品的 AI 工业设计工具 —— 上传一张参考图,三步生成专业级产品设计方案。支持扭扭车、滑板车、儿童三轮车、平衡车、宝宝餐椅等品类。

## 三步完成设计
1. **上传参考图** —— 放入一张你中意的设计灵感图
2. **选择产品类型与风格**(或上传手绘草图)
3. **一键生成** —— 得到高质量设计方案,可扩展三视图、局部微调

## 特点
- 参考图设计语言精准融合(造型 / 配色 / 质感)
- 专业 3D 写实渲染,高级质感
- 软萌 / 简约 / 运动 / 科技等多种风格,一键切换对比
- 从灵感到设计,几分钟搞定
